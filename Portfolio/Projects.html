NTP-Twitter-Webscraper

Webscraper built for the Northern Tornadoes Project. The Twitter webscraper was built using python, twitter API v2 and the tweepy library. Extracts relevant information from tweets searching by keyword. This info (the username of the tweeter, caption and tweet id) is then stored in an excel file that is created by the program and named by the user. This project was created for the Team 17's ES1050 winter design project for the Northern Tornadoes Project to use to collect weather data from Twitter.

BrainCanadaManagementSystem

Management system to match Brain Canada applicants to the reviewers in the most similar field. Uses Pandas library to convert .xlsx file to a readable dataframe for the applicants and the reviewers. Uses Levenshtein library to measure the difference in characters between the applicants and the reviewers primary research field. Then takes user input to select the applicant and tells the user the top 3 reviewers with the smallest amount of difference to that applicant's primary research field.
